# katana Slice Manager version 0.1.0

## Quick Start

### Requirements
- docker version >= 18.09.6
- docker-compose version >= 1.17.1

### Installation
`# ./install.sh`

### Start 
`# ./start.sh`

### Stop
`# ./stop.sh`
